# 📬 Contact Form with Flask Backend

## 📌 Project Description

This project is a simple **Contact Form Web Application** built using **HTML (frontend)** and **Python Flask (backend)**.
It allows users to submit their name, email, and message, which is then processed and stored.

---

## 🚀 Features

* User-friendly contact form
* Backend integration using Flask
* Stores messages in a file (`messages.txt`)
* Displays success message after submission
* Lightweight and easy to run

---

## 🛠️ Tech Stack

* HTML
* CSS (optional)
* Python
* Flask

---

## 📂 Project Structure

```
contact-form/
│── app.py
│── templates/
│     └── index.html
│── messages.txt
```

---

## ⚙️ How It Works

1. User fills the contact form
2. Data is sent to Flask backend
3. Backend processes the request
4. Data is stored in a text file
5. Success message is shown

---

## ▶️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/your-repo-name.git
```

2. Go to project folder:

```
cd your-repo-name
```

3. Install Flask:

```
pip install flask
```

4. Run the application:

```
python app.py
```

5. Open in browser:

```
http://127.0.0.1:5000/
```
